Need Help?

If you're having trouble using Harmony Care, please check the sections below:

  • To add products to cart, visit any product detail page and select your quantity.
  • If a product requires a prescription, you will be prompted to upload it during checkout.
  • Admins can log in via the login page and access product management.
  • If you encounter a blank screen or missing product, please ensure you're using the latest browser.

Accessibility

Harmony Care aims to be accessible to everyone:

  • All buttons are keyboard-navigable.
  • Form fields include labels for screen readers.
  • Colors and contrast have been adjusted for visibility.
  • We support resizing for small and large screens.

Common Errors

  • Login failed: Please ensure your username and password are correct.
  • Product not found: The product may have been removed or the ID is invalid.
  • Database connection error: Ensure your server is running and credentials are correct.
